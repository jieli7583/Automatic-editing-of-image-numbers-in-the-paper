类图语法

class 类名 定义类
+方法() : 返回值 公有方法
-字段 : 类型 私有字段
#字段 : 类型 受保护字段
«interface» 或 «singleton» 刻板标记
空行分隔类定义

关系（单独一行）：
A --> B : 标签 依赖/关联
A --* B : 标签 组合
A --o B : 标签 聚合
A --|> B 继承
A ..|> B 实现（虚线）
A ..> B : 标签 依赖（虚线）

时序图语法

participant 角色名 声明角色（可选）
A -> B : 消息 实线箭头
A --> B : 消息 虚线箭头（返回）
note : 文字 备注条
